# Paschal Omereife \| Full-Stack Developer Portfolio

![Portfolio
Preview](https://via.placeholder.com/1200x630?text=Paschal+Omereife+Portfolio)

A modern, luxurious full-stack developer portfolio showcasing technical
expertise, entrepreneurial journey, and interactive web applications.
Built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## 🚀 Live Demo

[View Live Portfolio] (https://paschalfidelporfolio.vercel.app)

## ✨ Features

### Core Pages

-   **Home Page** - Hero section with personal branding, skills
    showcase, and call-to-action
-   **About Me** - Detailed background, technical skills, and
    professional journey
-   **Projects Gallery** - Showcase of all web development projects
-   **Contact Page** - Contact form with validation and social media
    integration
-   **Business Page** - Entrepreneurial journey and business experience
-   **Gallery Page** - Personal moments and lifestyle gallery

### Interactive Tools (Projects)

1.  **Nigerian Mobile Number Validator** - Validate Nigerian phone
    numbers with regex
2.  **US Telephone Number Validator** - Validate US phone numbers with
    various formats
3.  **Palindrome Checker** - Check if words/phrases read the same
    forwards/backwards
4.  **Roman Numeral Converter** - Convert Arabic numbers to Roman
    numerals (1-3999)
5.  **Survey Form** - Interactive form with validation

### Technical Features

-   Fully responsive design (mobile, tablet, desktop)
-   Smooth scroll navigation with active section highlighting
-   Glass-morphism navigation bar
-   Luxury gold accent color scheme (#C5A267)
-   Bootstrap 5 components and grid system
-   Bootstrap Icons integration
-   Custom CSS animations and transitions
-   Form validation (client-side)
-   Interactive project cards with hover effects
-   Social media integration
-   Dynamic navigation active state based on scroll position

## 🛠️ Technologies Used

### Frontend

-   HTML5
-   CSS3 (Custom properties, Flexbox, Grid)
-   JavaScript (ES6+)
-   Bootstrap 5.3.0
-   Bootstrap Icons
-   Google Fonts (Inter)

### Tools & Libraries

-   Font Awesome (via Bootstrap Icons)
-   Custom regex patterns for validation
-   Intersection Observer for scroll effects

## 🔧 Installation & Setup

### Prerequisites

-   Modern browser
-   Local server (optional)

### Local Development

``` bash
git clone https://github.com/paschalfidel/portfolio.git
cd portfolio
```

Open index.html with Live Server or browser.
